The Packager

The Packager provides a collection of prebuilt packages, using Alpine's APK package format.

The packages are built using static-linking where possible, to minimise dependencies.

Supported architectures:

  • x86_64
  • aarch64

Usage

Use sudo if needed.

Add the public key

        curl -s https://thepackager.github.io/keys/packager@lists.alpinelinux.org-6368d779.rsa.pub -o /etc/apk/keys/packager@lists.alpinelinux.org-6368d779.rsa.pub

Add ThePackager as an APK repository

        sed -i -e "1i https://thepackager.github.io/alpine" /etc/apk/repositories
        apk update
